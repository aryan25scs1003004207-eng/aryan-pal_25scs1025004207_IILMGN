# 💊 AI Medicine Safety Checker

An AI-powered web application that helps users check whether a medicine is **Safe, Unsafe, or Requires Caution** based on inputs like age, disease, and interaction risk.

---

## 🚀 Project Overview

This project uses **Machine Learning (Decision Tree Classifier)** to predict medicine safety.  
It is designed as a real-life utility tool for basic healthcare awareness.

⚠️ **Note:** This is an advisory system and does NOT replace professional medical advice.

---

## 🧠 Features

- 🔍 Predicts medicine safety (Safe / Caution / Unsafe)
- 👤 Takes user inputs (Age, Disease, Interaction Risk)
- 💊 Includes commonly used medicines (India-focused)
- 🌐 Clean UI using Streamlit
- ⚡ Fast real-time predictions

---

## 🧪 How It Works

1. User selects:
   - Medicine
   - Age
   - Disease
   - Interaction Risk  

2. The trained ML model processes the input.

3. Output is displayed as:
   - ✅ Safe  
   - ⚠ Caution  
   - ❌ Unsafe  

---

## 📊 Machine Learning Model

- Algorithm: Decision Tree Classifier  
- Trained on:
  - Medicine data
  - Age conditions
  - Disease interactions
  - Risk levels
